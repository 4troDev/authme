# Authme

-   A simple 2FA app for desktop built with Electron and Node.js.

# How to use

-   Check out `extract/README.md`

# Build

`electron-packager . Authme --overwrite --platform=all --arch=x64 --icon=img/icon.icns --out=build`

# NPM Pacages used

### Dev

-   electron
-   electron-packager

### Main

-   bcrypt
-   create-desktop-shortcuts
-   cryptr
-   speakeasy
