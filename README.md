# Authme

-   A 2FA app for desktop built with Electron.

# Build

`electron-packager . Authme --overwrite --platform=all --arch=x64 --icon=img/icon.icns --out=build`

# NPM Pacages used

-   electron
-   electron-packager

-   bcrypt
-   cryptr
-   speakeasy
